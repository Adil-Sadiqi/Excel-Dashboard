**Sales & Profit Analysis Report**

**Overview**
This project provides a structured analysis of monthly profit and loss, category performance, and payment mode trends using the dataset `Details.xlsx`. The objective is to identify key business insights and highlight areas of improvement.

**Data Sources**
- **Details.xlsx**
  - **Monthly profit/loss** (January–December)
  - **Category-wise profit** (Furniture, Electronics, Clothing)
  - **Payment mode distribution** (Credit Card, Debit Card, COD, UPI, EMI)
  - **Sub-category breakdown** (Bookcases, Chairs, Tables, Furnishings)

## **Key Insights**
- **Monthly Trends**
  - Highest profit recorded in **February (8,322)**  
  - Losses observed in **August (-2,748)** and **December (-1,460)**  

- **Category Performance**
  - **Furniture**: 21,908 profit  
  - **Electronics**: 28,650 profit  
  - **Clothing**: 24,484 profit  
  - **Overall Net Profit**: 36,963  

- **Payment Modes**
  - **Debit Card** transactions contributed 11,612 in total amount.  
  - **Credit Card** and **COD** show mixed profit/loss outcomes.  

- **Sub-Categories**
  - **Bookcases**: +424 profit  
  - **Chairs**: +118 profit  
  - **Furnishings**: -70 loss  
  - **Tables**: -99 loss  

**Tools Utilized**
- **Microsoft Excel** (Pivot Tables, Charts, Data Cleaning)  
- **Structured categorization** for profit/loss analysis  

**Usage Instructions**
1. Open `Details.xlsx` in **Microsoft Excel**.  
2. Navigate to **pivot tables** for monthly and category analysis.  
3. Apply **filters** to view profit/loss by payment mode or sub-category.
